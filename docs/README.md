# Project Documentation

Вся проектная документация хранится в этой директории. Корневой `README.md` остается короткой точкой входа: быстрый старт, команды и ссылки на подробные документы.

## Документы

- [architecture.md](architecture.md) — структура приложения, backend/frontend modules, storage, queue и rendering flow.
- [api.md](api.md) — основные public/admin/system endpoints.
- [development.md](development.md) — локальная разработка, проверки, стиль и текущие conventions после cleanup.
- [performance.md](performance.md) — production build, стресс-тесты, RAM/CPU monitoring и рекомендации по производительности.
- [roadmap.md](roadmap.md) — актуальный статус и следующие этапы.

## Источники Правды

- Архитектурные решения и актуальная структура: `docs/architecture.md`.
- Что делать дальше: `docs/roadmap.md`.
- Команды разработки и проверки: `docs/development.md`.
- API surface: `docs/api.md`.
- Производительность и стресс-тесты: `docs/performance.md`.
